# DDS_example

## 軟體需求
1. Node.js [連結](https://nodejs.org/zh-tw/)
2. Git 

## 如何運行本專案?

1. 將本專案clone到本地端
2. 切換目錄至專案根目錄，執行`npm init`，此時npm會自動安裝所需要的套件
3. 執行`npm run start`，將會在本地端localhost:9000運行本系統
